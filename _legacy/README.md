# Digital Lost & Found Management System

A professional university campus Lost & Found system UI.

## Features
- **Report Lost/Found Items:** Easy to use forms for students and staff.
- **Browse Items:** Publicly accessible board of found items.
- **Smart Matching (UI):** Mock interface showing AI-powered matching confidence.
- **Claim System:** Process for claiming found items with proof of ownership.
- **Admin Dashboard:** For staff to verify matches and approve claims.

## Prerequisites
- Node.js installed on your machine.

## How to Run

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Start the Server:**
   ```bash
   node server.js
   ```

3. **Access the Application:**
   Open your browser and navigate to `http://localhost:3000`.

## Project Structure
- `server.js`: Express server handling routing.
- `public/`: Static assets (CSS, JS, Images, Data).
- `views/`: HTML pages.

## Technologies
- Node.js & Express
- HTML5, CSS3, Vanilla JavaScript
- Font Awesome (Icons)
